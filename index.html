Hasil Pekerjaan Selama PKL

Pengembangan Website Absensi Karyawan

Mendesain dan mengembangkan sistem absensi karyawan berbasis web dengan fitur pencatatan kehadiran, pengelolaan data karyawan, dan laporan absensi. Proyek ini menggunakan teknologi PHP, MySQL, dan JavaScript dengan dukungan Visual Studio Code (VS Code) sebagai text editor utama.

Sistem Pengelolaan Data Keluhan Pelanggan

Membangun sistem pengelolaan data keluhan pelanggan yang memungkinkan pengguna untuk menambahkan, melihat, mengedit, dan menghapus data keluhan secara interaktif. Sistem ini mengimplementasikan proses CRUD (Create, Read, Update, Delete) menggunakan PHP, MySQL, dan AJAX untuk mempercepat pengelolaan data secara real-time.

Sistem Login dan Hak Akses Pengguna

Mengembangkan sistem login yang dilengkapi dengan validasi keamanan dan akses berbasis peran (Role-Based Access Control). Proses ini memastikan bahwa hanya pengguna tertentu yang dapat mengakses halaman atau fitur tertentu. Proses ini menggunakan PHP untuk back-end, MySQL sebagai basis data, dan enkripsi password menggunakan password_hash().

Desain Antarmuka Pengguna (UI) yang Responsif

Membuat antarmuka website yang responsif dan dapat diakses dari berbagai perangkat, mulai dari desktop hingga smartphone. Proyek ini menggunakan kombinasi HTML, CSS, dan JavaScript dengan teknik Responsive Web Design (RWD) agar tampilan website tetap optimal di semua ukuran layar.

Perjalanan PKL

juli 2024

Awal Perjalanan PKL

Memulai Praktek Kerja Lapangan (PKL) di Telkom Renon Denpasar sebagai Desainer Web. Pada tahap ini, dilakukan pengenalan perusahaan, pengenalan tugas, dan penyiapan perangkat serta software pengembangan, seperti Visual Studio Code (VS Code) dan pengaturan lingkungan pengembangan berbasis PHP, JavaScript, dan MySQL.

Agustus 2024

Proyek Pertama: Sistem Absensi Karyawan

Mulai mengerjakan proyek pertama, yaitu pembuatan Sistem Absensi Karyawan. Proyek ini melibatkan pembuatan fitur login aman, pengelolaan data karyawan, dan pencatatan kehadiran. Proyek ini dibangun menggunakan PHP, MySQL, dan JavaScript serta menerapkan konsep CRUD (Create, Read, Update, Delete).

September 2024

Proyek Kedua: Sistem Pengelolaan Data Keluhan

Mengerjakan proyek kedua, yaitu Sistem Pengelolaan Data Keluhan. Proyek ini memungkinkan pengelolaan keluhan dari pelanggan secara real-time dengan fitur pencatatan, pencarian, pengeditan, dan penghapusan data. Proyek ini memanfaatkan AJAX untuk membuat halaman dinamis tanpa perlu memuat ulang halaman.

Oktober 2024

Peningkatan Desain Antarmuka (UI) dan Responsif

Meningkatkan tampilan antarmuka website dengan menerapkan prinsip Responsive Web Design (RWD) menggunakan kombinasi HTML, CSS, dan JavaScript. Tampilan website dioptimalkan agar dapat diakses dari berbagai perangkat, mulai dari desktop hingga perangkat seluler.

November - Desember 2024

Penyelesaian PKL dan Presentasi Laporan

Menyelesaikan semua proyek dan melakukan presentasi laporan akhir kepada pihak Telkom Renon Denpasar. Laporan ini mencakup seluruh proses pengembangan sistem, hasil proyek, serta pengalaman dan keterampilan yang diperoleh selama PKL. Presentasi ini merupakan bagian akhir dari proses PKL.
